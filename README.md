<h1>Challenge Ecriptador</h1>
Descripción : Aplicativo que permite encriptar un texto introduccido, descriptar el texto que dio como encriptación y copiar texto en ambos casos
Herramien.
<h2>Reglas</h2>

- La letra "e" es convertida para "enter"
- La letra "i" es convertida para "imes"
- La letra "a" es convertida para "ai"
- La letra "o" es convertida para "ober"
- La letra "u" es convertida para "ufat"

<h2>Recursos Gráficos </h2>  
El diseño e imagenes son públicos y fueron proporcionados por ALURA LATAM, para el desarrollo de este prouyecto.

<h2>Operación</h2>  
-  Teclear un en el campo de texto.
<br>
-  Para encriptar, dar clic en el botón encriptar. el resultado de la encriptación se visualizará en una columna del lado derecho
<br>
- Para desencriptar, dar cic en el botón de desencriptar, el resultado de la encriptación se visualizará en una columna del lado derecho
<br>
- El boton de copiar, copia el texto encriptado o desencriptado.
<br>
Esta aplicaicon sencilla es una base para poder trabajar con funciones, arregles y variales en javascript.
